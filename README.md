# ToggleNames Resurrected

`ToggleNames Resurrected` is a modernized Retail/Midnight-compatible continuation of the original [ToggleNames](https://www.curseforge.com/wow/addons/togglenames) addon by **Olafski**.

* [Wago Addons](https://addons.wago.io/addons/togglenames-resurrected)

## Credits

- **Olafski** — original ToggleNames addon (v1–v9, 2010–2016). The core idea, CVar names, and option categories all come from their work.

## What it does

It toggles character guild and title visibility based on context:
- Outside world
- Cities (resting)
- Party / Raid / Battleground / Arena

## Utility slash commands

- `/tnr` — show command help
- `/tnr debug` — toggle debug output (`ToggleNames.meta.debug`)
- `/tnr status` — print current context plus desired/current guild+title toggle state
- `/tnr update` — re-apply current title/guild visibility settings immediately
- `/tnr reset` — reset settings and reload UI

## Settings UI

Use the single modern panel:

**Settings → AddOns → ToggleNames Resurrected**

This panel includes:
- Guild visibility toggles per context (outside/city/party/raid/battleground/arena)
- Title visibility toggles per context
- Debug toggle
- Reset button
